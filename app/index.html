Trainings

  • Git Ninja Class

    2 Days 6th Sep 2014

    Git is a free and open source distributed version control system. It's invention was a blessing to the whole software world. However, it takes a bit of training to become really productive. Now is the best time to level up to become a real git Ninja!

    Why take this training?

    Have you ever wondered what the heck the HEAD is? And do you know how to properly do an interactive rebase without screwing everything up? And if you screwed it up, do you know how to restore things? Don't worry, you'll learn all this in this course. This course will make your average workday more productive. Every. Single. Day.

    What you will learn

    This is a quick run through the main topics which we will discuss. If you have questions not covered by the program, rest asured we don't leave any question unanswered!

    • Day 1 - Beginner Level
      • What is a distributed version control system (DVCS) anyway
      • How can DVCS help me or my company to become more productive
      • How to install and configure git
      • Git basics: init, clone, status, log, add, commit
      • How do I work with branches and tags
      • What are the different merge strategies
      • How do I resolve conflicts
    • Day 2 - Ninja Level
      • How to keep a history of semantic commits
      • How to auto-generate changelogs from git history
      • What is a rebase
      • Shootout: rebase workflow vs merge workflow
      • Performing an interactive rebase
      • Squashing and splitting existing commits
      • How to cherry pick commits
      • Make the reflog your friend
      • git internals

    What you will take home

    • A copy of the slides used to teach the course
    • Access to our private repository with lots of examples, exercises and git receipts
    • A bit of homework
    • thoughtram sticker
    • A six month micro plan for github.com - GitHub <3 thoughtram!

    Who is this training for?

    You can book the days individually. But you'll get the most out of it if you attend both days. This course is for you if...

    • Day 1
      • you have no experience with git but want to learn it
      • you are coming from SVN or other version control system
      • you know the basics of git but want to deepen your knowledge
    • Day 2
      • you are an intermediate git user and want to level up
      • you want to know all there is about partial adds and interactive rebases
      • you want to improve collaborative work among your team

    Where it happens

    When it happens

Code Of Conduct

All attendees, speakers, sponsors and volunteers at our trainings, workshops or webinars are required to agree with the following code of conduct. Organisers will enforce this code throughout all events. We are expecting cooperation from all participants to help ensuring a safe environment for everybody.

tl;dr: Don't be a Jerk

The Quick Version

Our trainings, workshops or webinars are dedicated to providing a harassment-free experience for everyone, regardless of gender, sexual orientation, disability, physical appearance, body size, race, or religion. We do not tolerate harassment of our participants in any form.

Sexual language and imagery is not appropriate for any event venue, including talks, workshops, parties, Twitter and other online media. Participants violating these rules will be sanctioned or expelled from the event without a refund at the discretion of the event organisers.

The Less Quick Version

Harassment includes offensive verbal comments related to gender, sexual orientation, disability, physical appearance, body size, race, religion, sexual images in public spaces, deliberate intimidation, stalking, following, harassing photography or recording, sustained disruption of talks or other events, inappropriate physical contact, and unwelcome sexual attention.

Participants asked to stop any harassing behavior are expected to comply immediately.

If a participant engages in harassing behavior, the event organizers may take any action they deem appropriate, including warning the offender or expulsion from the event with no refund.

If you are being harassed, notice that someone else is being harassed, or have any other concerns, please contact us immediately.

We expect participants to follow these rules at workshop venues and workshop-related social events.
